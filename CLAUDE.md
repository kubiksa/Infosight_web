# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

Single-page static consulting website for a data engineer and cloud architect.

## Stack Constraints

- **Plain HTML and CSS only** — no frameworks, no npm, no build tools
- **No JavaScript** unless absolutely necessary for core functionality
- **No package.json, node_modules, or any build artifacts**
- File structure is strictly `index.html` and `styles.css`

## Development

Open `index.html` directly in a browser — no build step, no dev server needed.

## Design Principles

- Clean, professional, and minimal aesthetic
- All styling lives in `styles.css`; no inline styles
- No external CSS frameworks (no Bootstrap, Tailwind, etc.)
- No CDN-loaded scripts or style sheets unless essential
